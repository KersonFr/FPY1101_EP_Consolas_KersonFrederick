# Sistema de Gestión de Consolas de Videojuegos

## Descripción
Este es un programa en Python desarrollado para administrar el catálogo y control comercial de consolas de videojuegos mediante el uso de diccionarios sincronizados.

## Funcionalidades
- Agregar consola (con validaciones estrictas)
- Buscar consola por sigla
- Eliminar consola (sincronizado)
- Mostrar todas las consolas (recorrido cruzado)

## Cómo ejecutar
Ejecutar el archivo principal desde la terminal con el comando:
python consolas.py
